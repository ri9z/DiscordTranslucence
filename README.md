# Requirements:
- [BetterDiscord](https://betterdiscord.app/) or [Vencord](https://vencord.dev) 
- SF Pro Display font

# Installation:
- Copy `TranslucencePlus.theme.css` to your Vencord/BetterDiscord theme folder.
- Enable the theme in DB's themes settings.
